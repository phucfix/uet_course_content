## Đề bài Final Project

Mục tiêu: Áp dụng toàn bộ kiến thức đã học để xây dựng một ứng dụng nhỏ (CLI hoặc web).

Yêu cầu tối thiểu:

- Tài liệu hướng dẫn sử dụng và mã nguồn trên GitHub
- Tests hoặc hướng dẫn manual testing
- Hướng dẫn deploy (nếu là web)

Gợi ý đề tài:

- Ứng dụng quản lý công việc (To-do list) với lưu trữ file
- Web nhỏ với form, validation và lưu vào SQLite
- Trò chơi console đơn giản (rắn săn mồi, tic-tac-toe)

Tiêu chí chấm điểm:

- Tính đúng đắn và ổn định
- Chất lượng code và cấu trúc repository
- Documentation và hướng dẫn chạy

Nộp bài: Push repository lên GitHub và gửi link vào hệ thống nộp bài.
