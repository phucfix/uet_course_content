## Week 0 — Giới thiệu và Thiết lập môi trường

Mục tiêu

- Hiểu vai trò của database trong ứng dụng
- Cài đặt PostgreSQL hoặc SQLite cho thực hành

Nội dung

- Khái niệm bảng, rows, columns, và schema
- Cài đặt môi trường và kết nối cơ bản

Bài tập

1. Cài PostgreSQL hoặc sử dụng SQLite và tạo database mẫu
2. Viết vài câu lệnh SQL cơ bản (SELECT, INSERT)
