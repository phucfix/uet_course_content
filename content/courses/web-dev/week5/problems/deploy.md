Follow steps to deploy your static site to GitHub Pages.

## Getting Started

Log into [cs50.dev](http://localhost:3000/auth/github?returnTo=http%3A%2F%2Flocalhost%3A5173%2Fcourses%2Fweb-dev%2F%3FenrollCourseSlug%3Dweb-dev%26enrollWeek%3D5%26openSubmit%3D1%26problem%3Ddeploy) using your GitHub account and click Authorize cs50, then close the tab.

## How to check / Cách kiểm tra

Chạy:

```bash
check50 --local uet36/problems/2025/web/deploy
```

## How to submit / Cách nộp

Chạy:

```bash
submit50 --local uet36/problems/2025/web/deploy
```
