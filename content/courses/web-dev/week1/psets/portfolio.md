# Personal Portfolio

Xây dựng một trang portfolio cá nhân sử dụng HTML và CSS.

## Yêu cầu

1. **Header**: Navigation bar với menu
2. **Hero Section**: Giới thiệu về bản thân với hình ảnh
3. **About Section**: Thông tin về kỹ năng và kinh nghiệm
4. **Projects Section**: Hiển thị các dự án (có thể dùng placeholder)
5. **Contact Section**: Form liên hệ
6. **Footer**: Thông tin liên hệ và social media links

## Yêu cầu CSS

- Responsive design (mobile, tablet, desktop)
- Sử dụng Flexbox hoặc Grid
- Màu sắc và typography đẹp mắt
- Hover effects

## Gợi ý

- Sử dụng semantic HTML5 tags
- Tổ chức CSS theo BEM methodology
- Sử dụng CSS variables cho colors
- Thêm animations cho các elements
