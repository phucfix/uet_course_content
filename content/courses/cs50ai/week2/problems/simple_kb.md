Create a small knowledge base and implement simple queries to retrieve facts.

## Getting Started

Log into [cs50.dev](http://localhost:3000/auth/github?returnTo=http%3A%2F%2Flocalhost%3A5173%2Fcourses%2Fcs50ai%2F%3FenrollCourseSlug%3Dcs50ai%26enrollWeek%3D2%26openSubmit%3D1%26problem%3Dsimple_kb) using your GitHub account and click Authorize cs50, then close the tab.

## How to check / Cách kiểm tra

Chạy:

```bash
check50 --local uet36/problems/2025/ai/simple_kb
```

## How to submit / Cách nộp

Chạy:

```bash
submit50 --local uet36/problems/2025/ai/simple_kb
```
