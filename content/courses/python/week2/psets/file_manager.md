# File Manager

Tạo một chương trình quản lý file cho phép người dùng đọc, ghi và xử lý file.

## Yêu cầu

1. Menu cho phép:
   - Đọc file
   - Ghi file
   - Tìm kiếm trong file
   - Đếm số từ trong file

2. Xử lý lỗi khi file không tồn tại

## Gợi ý

- Sử dụng `try-except` để xử lý lỗi
- Sử dụng functions để tổ chức code
- Sử dụng `with` statement khi làm việc với file
