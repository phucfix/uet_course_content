Log into [cs50.dev](https://api.phucnguyen.xyz/auth/github?returnTo=https%3A%2F%2Fphucnguyen.xyz%2Fcourses%2Fpython%2F%3FenrollCourseSlug%3Dpython%26enrollWeek%3D5%26openSubmit%3D1%26problem%3Dwrite_tests) using your GitHub account and click Authorize cs50, then close the tab.
