Các thẻ HTML quan trọng

  • <h1> đến <h6>: Tiêu đề
  • <p>: Đoạn văn
  • <a href="">: Liên kết
  • <img src="" alt="">: Hình ảnh
  • <ul>, <ol>, <li>: Danh sách
  • <div>: Container
  • <span>: Inline container

    margin: 10px;
}
